1. Install SandboxieInstall.exe (autolif-master\SandboxieInstall.exe).
2. Browse files and navigate to LiF's root folder (e.g. C:\Program Files (x86)\Steam\steamapps\common\L̲i̲f̲e̲ ̲i̲s̲ ̲F̲e̲u̲d̲a̲l̲ ̲M̲M̲O̲)
